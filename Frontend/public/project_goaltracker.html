Project Overview

Introduction

Goal Tracker is a full-stack MERN application designed to help users manage their daily tasks efficiently. This project enables users to log in, add, and delete their to-dos once completed, ensuring better productivity and organization. The frontend is built using Vite + React.js for a fast and smooth user experience, while the backend, powered by Node.js, Express.js,and MongoDB, ensures secure data management and seamless API communication.

Features

- User Authentication : Secure login and registration system.
- Add To-Do Items : Users can create and save their tasks.
- Delete Completed Tasks : Remove tasks once completed.
- Real-Time Updates : Instant changes reflected in the UI.
- Responsive UI : Optimized for all devices.

Technology Stack

Frontend: React.js (Vite), Tailwind CSS for styling
Backend: Node.js, Express.js
Database: MongoDB (NoSQL)
Authentication: JWT (JSON Web Token)
Deployment:
Frontend: Vercel
Backend: Render

Deployment & Hosting

The frontend is deployed on Vercel, ensuring fast load times and seamless performance. The backend API is hosted on Render, ensuring a reliable and scalable server infrastructure.

Conclusion

Goal Tracker is a powerful task management tool that helps users stay productive by keeping track of their daily goals. This project demonstrates my ability to build a full-stack web application, integrate a secure backend, and deploy a fully functional MERN-based project.
